# Daikibo-Factory-Telemetry-Analysis-
Analyzed IoT telemetry data from Daikibo's four manufacturing facilities to identify machine downtime patterns. Built an interactive Tableau dashboard to visualize downtime by factory and device type, helping identify locations and machines responsible for the highest operational disruptions.

## Objective
Analyze IoT telemetry data from 4 factories to identify:
- Factory with highest downtime
- Device types causing most downtime

## Tools
- Tableau
- JSON Data
- Data Visualization

## Dashboard
Created interactive Tableau dashboard with:
- Down Time per Factory
- Down Time per Device Type
- Factory filter interaction

## Insights
Identified the factory with maximum downtime and the devices contributing most to failures.
